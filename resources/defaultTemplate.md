# Welcome to Markdown Viewer

This is a simple, clean markdown editor and viewer for macOS.

## Features

- **Edit Mode**: Focus on writing your markdown
- **Preview Mode**: See your rendered markdown
- **Split Mode**: Edit and preview side-by-side

## Getting Started

### Basic Formatting

You can use standard Markdown syntax:

- **Bold text** with `**text**`
- *Italic text* with `*text*`
- `Inline code` with backticks
- [Links](https://example.com) with `[text](url)`

### Lists

Unordered lists:
- Item 1
- Item 2
- Item 3

Ordered lists:
1. First item
2. Second item
3. Third item

### Code Blocks

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

### Blockquotes

> This is a blockquote.
> It can span multiple lines.

## Keyboard Shortcuts

- **Open File**: `Cmd+O`
- **Save**: `Cmd+S`
- **Save As**: `Cmd+Shift+S`
- **Quit**: `Cmd+Q`
